# Milestone 11: X Launch

## Goal

Prepare Agent Ready for an initial X.com launch with a clear message, useful demo asset, and a
marketing subagent role that can keep future promotion aligned with the product.

## Scope

- Add a Marketing Agent role definition.
- Draft launch positioning, X copy, thread structure, and visual/demo guidance.
- Keep claims accurate to the current CLI: local-first scan, safe init, prescription generation,
  no API keys, and no hosted AI calls.

## Non-Goals

- Do not add analytics, tracking, paid campaigns, or automated posting.
- Do not imply Agent Ready fixes code, runs an AI model, uploads repositories, or replaces coding
  agents.

## Acceptance Criteria

- The marketing role explains audience, responsibilities, inputs, output format, and safety rules.
- Launch copy includes a concise primary post, a thread outline, visual plan, and CTA.
- The message can be posted without overclaiming the current MVP.

## Validation

- Review wording against README and landing page.
- Keep all generated files plain Markdown.
